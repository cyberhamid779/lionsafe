# CyberShield AZ — Layihə Sənədi

## Problem
Azərbaycanda 2024-cü ildə bank fırıldaqçılığından **22 milyon AZN** zərər verildi.
Hücumların 70%-i phishing-dir. Bankların nə əməkdaşları bu hücumları tanıyır,
nə də müştərilər nə edəcəyini bilir.

---

## Həll
Azərbaycan dilində, bank sektoruna uyğun kibertəhlükəsizlik platforması.

### Modullar
1. **Phishing Simulasiya + Təlim** — əməkdaşlara fake hücum göndər, nəticəyə görə fərdi təlim
2. **Müştəri Bildiriş Sistemi** — şübhəli zəng/mesaj aldıqda bank xəbərdar olsun
3. **Risk Skorinq** — hansı əməkdaşın risk altında olduğunu göstərən dashboard
4. **Incident Response Avtomatlaşdırma** — hadisə baş verdikdə avtomatik addımlar

---
